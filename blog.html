What is the Document Object Model (DOM)?

DOM is an Object model of HTML file. It's a programming interface for HTML document or file. When the browser loads, HTML file get converted into DOM tree. JavaScript can easily manipulate HTML elements using DOM.

How do you select an element from the DOM?

There are 5 methods to select or read an element from the DOM
1. getElementsByTagName()
it selects all the elements with the specific tag name(div, p, span, etc.) & returns an HTMLCollection(array-like object)
2. getElementById()
it selects an element by its unique id attribute.
3. getElementsByClassName()
it selects all elements with a specific class name & returns an HTMLCollection(array-like object)
4. querySelector()
it selects only the first element that matches a CSS selector(class, id, attribute, etc.)
5. querySelectorAll()
it selects all elements that match a CSS selector(class, id, attribute, etc.) & returns a NodeList.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where we delegate event handling to a parent element rather than adding event listeners to individual child elements. Instead of attaching event listeners to multiple elements, we attach one event listener to a common ancestor (often a container), and take advantage of event bubbling to catch events that occur on its descendants.
Event Delegation is very useful because-
1. Improves performance: Attaching individual event listeners to many elements can consume resources, especially in large lists or dynamic content. With event delegation, a single listener on the parent can manage events for all child elements, reducing memory usage and improving efficiency.
2. Handles dynamic elements: Event delegation allows us to handle events on elements that are dynamically added to the DOM after the page has loaded. Since the parent element already has the event listener, it can respond to events on newly added child elements without the need to add new listeners.
3. Simplifies Code: Instead of managing many listeners, a single listener at a higher level can manage all events, reducing the complexity of our code.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, we use setAttribute() to change or add an attribute and removeAttribute() to remove it. We can access and modify an element's style by using the style property, like: element.style.color = 'blue' to change its color. To retrieve an attribute's value, we use getAttribute(). For class manipulations, we can use classList methods like: add(), remove(), or toggle().
